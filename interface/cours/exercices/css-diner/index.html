Pas d'inquiétude, vous allez y arriver!

Vous vous apprêtez à apprendre les sélecteurs CSS! Un sélecteur permet de choisir à quel(s) élément(s) on souhaite appliquer une règle de style

Exhibit 1 - A CSS Rule

p {
   margin-bottom: 12px;
}

Ici "p" est le sélecteur, il sélectionne tous les paragraphes, c'est-à-dire tous les éléments <p> et leur applique 12px de margin-bottom

Pour jouer, taper voter sélecteur CSS dans l'éditeur ci-dessous pour sélectionner l'objet voulu sur la table. Si vous y arrivez, vous accéderez au niveau suivant

Passez votre souris sur les objets sur la table pour voir leur balise HTML

Vous trouverez toute l'aide nécessaire sur la droite →
