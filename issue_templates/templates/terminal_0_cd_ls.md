## Objectif

Maitriser les commandes de base dans le terminal.

1/ Ouvrir un terminal

- [ ] Tapez ctrl + alt + T pour ouvrir un terminal sous Linux

2/ Commande `ls`

- [ ] Tapez la commande `ls` pour lister les fichiers et dossiers dans le répertoire courant.
- [ ] Tapez la commande `ls -l` pour afficher les fichiers et dossiers avec des détails supplémentaires (permissions, taille, date de modification).
- [ ] Tapez la commande `ls -a` pour afficher tous les fichiers, y compris les fichiers cachés (ceux qui commencent par un point).

3/ Commande `cd`

- [ ] Tapez la commande `cd nom_du_dossier` pour naviguer dans un dossier spécifique.
- [ ] Tapez la commande `ls` pour vérifier que vous êtes bien dans le dossier souhaité.
- [ ] Tapez la commande `cd ..` pour revenir au dossier parent.
- [ ] Tapez la commande `ls` pour vérifier que vous êtes bien revenu au dossier parent.

4/ Commande mkdir

- [ ] Tapez la commande `mkdir nom_du_nouveau_dossier` pour créer un nouveau dossier.
- [ ] Tapez la commande `ls` pour vérifier que le nouveau dossier a bien été
- [ ] Tapez la commande `cd nom_du_nouveau_dossier` pour naviguer dans le nouveau dossier.
- [ ] Tapez la commande `ls` pour vérifier que vous êtes bien dans le nouveau dossier (il doit être vide).

5/ Commande touch

- [ ] Tapez la commande `touch nom_du_fichier.txt` pour créer un nouveau fichier texte.
- [ ] Tapez la commande `ls` pour vérifier que le nouveau fichier a bien été créé.

6/ Commande nano

- [ ] Tapez la commande `nano nom_du_fichier.txt` pour ouvrir le fichier texte dans l'éditeur nano.
- [ ] Écrivez du texte dans le fichier.
- [ ] Enregistrez et quittez nano en appuyant sur `CTRL + X`, puis `Y` pour confirmer l'enregistrement, et enfin `ENTER` pour valider le nom du fichier.

7/ Commande cat

- [ ] Tapez la commande `cat nom_du_fichier.txt` pour afficher le contenu du fichier texte dans le terminal.

8/ Commande rm

- [ ] Tapez la commande `rm nom_du_fichier.txt` pour supprimer le fichier texte.
- [ ] Tapez la commande `ls` pour vérifier que le fichier a bien été supprim
